About us

Mission & Values

We are a Dublin-based real estate team focused on helping people find high-quality and affordable homes across Ireland.

This app was created as part of a college assignment to demonstrate property listings, detail views, and booking features using modern web technologies like Ionic and Firebase.

Our goal is to make the process of renting or buying as simple as possible ‚Äì from discovering properties to viewing details to booking a viewing. Everything is powered by Firebase and Ionic.

What this app does

  • Lists available properties fetched from Firebase Firestore
  • Shows full details and photos for each property
  • Allows users to register and log in using Firebase Auth
  • Lets users browse featured properties and see similar ones
  • Saves viewing requests directly to Firestore
  • Fully responsive design for mobile, tablet, and desktop
